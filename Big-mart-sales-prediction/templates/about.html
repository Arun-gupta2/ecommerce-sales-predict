About the Dataset

The dataset used for the Big Mart Sales Prediction project consists of transactional and product data collected from various retail outlets. This data helps predict sales across multiple outlets based on product attributes, pricing, and store characteristics.

Main Features of the Dataset

  • Item Identifier: Unique ID for each product.
  • Item Weight: Weight of the item in kilograms.
  • Item Fat Content: Describes if the item is low-fat or regular.
  • Item Type: Categorization of the product, e.g., Dairy, Snacks, etc.
  • Item MRP: Maximum Retail Price of the product.
  • Outlet Identifier: Unique ID for each store.
  • Outlet Location Type: Classification of the store's location (Tier 1, 2, or 3).
  • Outlet Type: Store category (e.g., Grocery Store, Supermarket).
  • Outlet Age: Age of the outlet in years.

By using this data, we aim to predict future sales based on key product attributes, location details, and store type. These predictions can help retail managers optimize stock levels, adjust prices, and maximize revenue.
